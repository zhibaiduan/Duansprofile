## What It Is

A Data Science final course project at HTW Berlin. The goal: build a machine learning model that predicts whether a person has a sleep disorder — insomnia, sleep apnea, or none — based on health indicators like heart rate, blood pressure, occupation, and sleep duration.

The project focused on how to improve model performance through feature engineering, feature selection, and hyperparameter tuning — and how to evaluate the results rigorously.

## Pipeline

Five stages, each a deliberate decision point:

1. **Data Understanding & EDA** — Explore data quality, distributions, and relationships between features and the target variable.
2. **Feature Engineering & Preprocessing** — Create new features, select the most useful ones, encode and scale for modelling.
3. **Model Selection & Benchmarking** — Benchmark 5 models with cross-validation to identify the top candidates.
4. **Hyperparameter Fine Tuning** — Use GridSearchCV to optimize the shortlisted models.
5. **Final Evaluation** — Evaluate on the held-out test set using accuracy, confusion matrix, ROC/AUC, and bootstrapping.
